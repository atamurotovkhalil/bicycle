# Bicycle E-commerce Website

A modern and responsive **bicycle e-commerce web application** built with **React, Vite, and Tailwind CSS**, focusing on performance, clean UI, and a smooth shopping experience.

---

##  Project Overview

This project is a frontend-focused e-commerce platform designed for selling bicycles online..

The application emphasizes **component reusability**, **state management**, and **performance optimization** using modern React tooling.

---

##  Features

* Product listing and product detail pages
* Interactive UI components (accordion, sliders, switches)
* Smooth navigation with React Router
* Global state management using Zustand
* Modern UI/UX with Tailwind CSS and Radix UI
* Fast development and build performance with Vite

---

##  Tech Stack

### Frontend

* **React 18**
* **TypeScript**
* **Vite**
* **Tailwind CSS**
* **React Router**
* **Zustand** (state management)

### UI & Styling

* Radix UI
* Lucide React / React Icons
* Tailwind Merge & Tailwind Animate
* Embla Carousel

### Development Tools

* ESLint
* PostCSS & Autoprefixer
* TypeScript ESLint
* Vite Plugin React
